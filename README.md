## Run Locally

Clone the project

```bash
  git clone https://github.com/blake-c-aggienetwork/adventOfCode2022
```

Navigate to the desired puzzle 

```bash
  cd adventOfCode2022/01
```

Compile the code-- I am using clang for this, update the $(CC) in the makefile to change

```bash
  make
```

Run the executable

```bash
  ./main.out
```
